🎬 SFF Semantic Analysis Laboratory

Computational Modeling for Movie Festival Recommendation and Scheduling

1. Project Overview

This laboratory demonstrates how to resolve the "Information Overload" and "Cold Start" problems during the 27th Shanghai International Film Festival. As a Mathematics and Applied Mathematics student, I developed this system to bridge the gap between abstract cinematic content and practical decision-making using high-dimensional vector spaces.

2. Interactive Lab (Live Demo)

Enter a movie title (e.g., Blue Velvet or Pride and Prejudice) below to see the semantic analysis in real-time.

3. Mathematical Methodology

The system maps film synopses into a 384-dimensional vector space using Sentence-BERT (S-BERT). Similarity is calculated via Cosine Distance, measuring the angular proximity between concepts:

$$\text{similarity} = \cos(\theta) = \frac{\mathbf{A} \cdot \mathbf{B}}{\|\mathbf{A}\| \|\mathbf{B}\|}$$
